# 🎮Tic Tac Toe

A simple **Tic Tac Toe game built using Java**, available in two versions:

* **Console Version** – Playable in the terminal.
* **GUI Version** – Built using Java Swing for a graphical interface.

## Features

* Two-player gameplay
* Win and draw detection
* Input validation
* Console and GUI implementations

## Technologies

* Java
* Java Swing (GUI)
